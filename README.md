# Dev Portfolio

Premium, fully responsive web developer portfolio built with plain HTML, CSS and JavaScript — no build step, no dependencies. Just open `index.html` in a browser or deploy to GitHub Pages.

## Features

- Dark futuristic theme with light-mode toggle
- Smooth scroll-reveal animations
- Typing hero animation with particle background
- Glassmorphism UI elements
- Project filtering with smooth transitions
- Animated contact form
- Transparent pricing with add-ons
- Mobile-first responsive design
- SEO meta & Open Graph tags
- Cross-browser compatible

## Run locally

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Structure

```
index.html        # all sections (navbar, hero, about, skills, projects, pricing, contact, footer)
style.css         # design system, dark/light theme, responsive layout
script.js         # typing, particles, scroll reveal, filters, theme toggle
assets/img/       # project images & profile photo (add your own)
```

## Deploy to GitHub Pages

1. Push to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main** branch
4. Your site will be live at `https://<username>.github.io/<repo>/`
